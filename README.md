# 📝 To-Do List Web App

A user-friendly to-do list that allows users to:

- ✅ Add tasks with title and scheduled date/time
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 🕒 View timing in **AM/PM** format
- 🎨 Enjoy a beautiful, warm UI design
